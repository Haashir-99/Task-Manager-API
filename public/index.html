Welcome to the Task Manager API

This API provides robust features for managing tasks, teams, and roles.

It includes full CRUD operations for:

  • Tasks: Create, Read, Update, and Delete tasks.
  • Teams: Manage teams with full CRUD functionality.
  • Role-based Access: Assign roles and permissions to team members for enhanced security and control.
